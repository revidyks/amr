===============================================================================
                       amr (automatic mail registration)                       
===============================================================================

Программа для регистрации почтовых аккаунтов на mail.ru

-------------------------------------------------------------------------------

Текущие возможности

 - работает с сервисом antigate.com (для автоматического разгадывания капч) :)
 - работает через прокси серверы 
 - домен выбирается пользователем любой из доступных на mail.ru
 - подбирает имена и фамилии по словарям 
 - выбор даты рождения случайным образом в указанном диапазоне лет
 - на основании имени, фамилии и даты рождения генерируется уникальный логин
 - пароль генерируется случайным образом 
 - все зарегистрированные акаунты записываются в текстовый файл

-------------------------------------------------------------------------------

Планы на будущее

 В данный момент работаю над многопоточностью и активацией у зарегистрированных 
 аккаунтов сервиса "Мой Мир". Дальше планирую добавить другие почтовые серверы.

-------------------------------------------------------------------------------

Описание поставки

 - amr.exe       - основной файл программы (его нужно запускать)
 - settings.txt  - файл основных настроек программы
 - proxy.txt     - файл для хранения списков прокси серверов
 - familys_f.txt - словарь женских фамилий
 - familys_m.txt - словарь мужских фамилий
 - names_f.txt   - словарь женских имен
 - names_m.txt   - словарь мужских имен
 - RaedMe.txt    - описание программы, которое вы читаете сейчас

-------------------------------------------------------------------------------

Файлы настроек программы

 settings.txt

  в этом файле должна быть заполнена как минимум первая строка
  в ней Вы должны указать Ваш captcha ключ

  во второй строке можно указать минимальный год
  в третьей строке можно указать максимальный год

 proxy.txt

  нужно заполнять если вы собираетесь работать через прокси
  пример заполнения есть в файле
  обязательны для заполнения первые два параметра (сервер и порт)
  остальные можно оставить пустыми
  каждый прокси сервер нужно указывать в новой строке
  при работе программа будет выбирать сервер случайным образом каждый раз

 familys_f.txt, familys_m.txt, names_f.txt, names_m.txt

  можно дополнять или изменять при необходимости
  на каждой строке по одному слову
  данные выбираются случайным образом

-------------------------------------------------------------------------------

Начало работы

 в верхнем левом поле необходимо указать количество аккаунтов
 выбор пола в данный момент не доступен (принудительно женский)
 в нижнем левом поле необходимо выбрать минимум один домен
 для работы через прокси нужно установить галку "use proxy"
 галка "my world" в данный момент не работает

 после указания всех необходимых данных нужно нажать кнопку "start"

-------------------------------------------------------------------------------

Выходные данные программы

 зарегистрированные аккаунты записываются в файл list_akk.txt
 в формате mail;password

 в процессе работы может создаваться файл error_log.txt
 в него будут записываться ошибки
